Modules


Crear Modulo

Para crear un modulo, se define una funcion y se emplea la palabra reservada export default. El nombre del archivo es nuevaFuncion.js

            
export default function(message){
alert(message);
}

Se puede exportar aplicaciones enteras, solamente se debe introducir la palabra clave export. 

            
function sumTwo(a, b) {
    return a + b;
}
function sumThree(a, b, c) {
    return a + b + c;
}
let api = {
    sumTwo,
    sumThree
};
export default api;
Importar un Modulo

Para importar un archivo externo, dentro del archivo en el que estamos trabajando, se utiliza la palabra import seguido de un identificador y la ruta de la funcion que deseamos importar. El nombre del archivo es miapp.js 

            
import datos from './nuevaFuncion';
datos("ECMASCRIPT");

Se puede importar nombre de objeto y renombrarlos dentro del archivo destino. 

            
import {
    sumTwo as addTwoNumbers,
    sumThree as sumThreeNumbers
} from 'math/addition';
