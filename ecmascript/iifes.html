IIFES and Blocks


Alcance en Funciones

Las variables locales declaradas utilizando la palabra clave "var" están en el ámbito de la función que la declara. Si no existe tal función, las variables se crean como variables globales, contaminando así el alcance global. Los IIFE ayudan a limitar el alcance de las variables locales. 

            
(function() {
    var oracion = "hola mundo";
    
})();
console.log(oracion); // oracion is not defined
Alcance en Bloques

Utilizar variables de bloque de ámbito como "let y const", de tal manera que el alcance de dichas variables sea a nivel del bloque que las declara. 

            
{
    let oracion = "hola mundo";
};
console.log(oracion); // oracion is not defined
IIFES

Ejemplo utilizando funciones iife. Se invoca a tres funciones con variables locales,el resultado de la funcion contar, es un numero entero incremental. 

            
let contar = (function() {
    let contador = 0;
    return function() {
        ++contador;
        return `${contador}`;
    };
})();

console.log(contar()); // "1"
console.log(contar()); // "2"
console.log(contar()); // "3"
BLOQUES

Ejemplo utilizando bloques. Se crea una variable asignando el valor de una funcion , dentro del retorno se enumeran los metodos que puede utilizar la funcion principal. 

            
let contar = (function() {
    let contador = 0;  
    return {
        adicion() {
            ++contador;  
        },  

        get valor() {
            return contador;
        }
    };
})();
contar.adicion();
console.log(contar.valor); // 1
contar.adicion();
console.log(contar.valor); // 2
contar.adicion();
console.log(contar.valor); // 3
