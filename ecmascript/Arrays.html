Arrays


Declaracion e Inicializacion

En ES5 los arrays se inicializan de acuerdo a un apuntador, que depende del tamaño del array. Si se quiere asignar algun valor dentro de un vector a una variable, se realiza de forma explicita solicitando el apuntador del valor del array.

            
let codigo= [125645, 123548, 669555, 154522];
let cod1= codigo[0];
let cod2= codigo[1];
let cod3= codigo[2];
let cod4= codigo[3];
console.log(cod1,cod2,cod3,cod4);

En ES6 existe la capacidad de asignar por destructuring que hace posible la extracción de datos de arreglos u objetos usando una sintaxis que equivale a la construcción de arreglos y objetos literales.

            
let codigo= [125645, 123548, 669555, 154522];
let [cod1,cod2,cod3,cod4]= codigo;
console.log(cod1,cod2,cod3,cod4);

Se pueden suprimir elementos del array que no se quieran asignar.

  
 let codigo= [125645, 123548, 669555, 154522];
let [cod1, , ,cod2]= codigo;
console.log(cod1,cod2);
Destructuring en Funciones

En ES6 tambien afecta en la utilizacion de funciones con array, donde la destructuring se asigna a la funcion que devuelve el array. 

            
function fn1(){
let codigo= [125645, 123548, 669555, 154522];
return codigo;
}
let [cod1,cod2,cod3,cod4]=fn1();
console.log(cod1,cod2,cod3,cod4);

function fn1(){
let codigo= [125645, 123548, 669555, 154522];
for(let recorrer of codigo){
console.log( recorrer );
