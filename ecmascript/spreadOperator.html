Spread Operator


Arrays

La utilizacion del operador de propagacion (spread operator) en arrays, permite a√±adir, invocar atributos de un arrays dentro de otro arrays mas facilmente, sin la utilizacion de metodos como push, splice, concat.

La forma de declara un spread operator es con tres punto seguidos de un identificador.

            
let numeros=[1,2,0,3];
let vect=[...numeros, 4, 5, 6]; 
console.log(vect); // 1,2,0,3,4,5,6
Metodo Push

La creacion de los operadores de propagacion permiten que algunos metodos se optimicen. Un ejemplo es el metodo push, que reduce el numero de comandos para ejecutar la operacion push

            
let arr1 = [0, 1, 2];
let arr2 = [3, 4, 5];
arr1.push(...arr2);
console.log(arr1); // 0,1,2,3,4,5

  return n1+" "+n2+" "+n3+" "+n4;
}

console.log(suma(5,6,...numbers));// "5 6 12 5"
console.log(suma(5,...numbers)); // "5 12 5 8"
