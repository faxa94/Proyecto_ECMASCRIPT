Parameters


Parametros por Defecto

En ES5 se declaran los parametros por defecto de la siguiente manera:

            
function sumas(x, y) {
    x = x || 0;
    y = y || 0;
    return x + y;
}

En ES6 se utiliza el siguiente formato:

            
function sumas(x=0, y=0) {
    return x + y;
}
sumas(5, 5); // 10
sumas(33); // 33
sumas(); // 0
Parametros Rest

En ES5 cuando no sabemos cual es el tamaño de un arrays se define el uso de metodos como length. 

  
function fn1(valor) {
    for (var i=0; i <= valor.length; i++) {
        console.log(valor[i]);
    }
}

En ES6 el paso de parametros indefinidos se utiliza el comando of dentro del loop. 

  
function fn1(...valor) {
    for (let n of valor) {
        console.log(n);
    }
}
Parametros Named

Uno de los patrones en ES5 para manejar los parámetros con nombres, era la utilizacion de las opciones del objeto modelo, adoptado de jQuery. 

  
function fn1(tam) {
    var height = tam.height || 250;
    var width  = tam.width  || 250;
    var lineStroke = tam.lineStroke || 'green';
}

Se puede realizar el codigo anterior pero aplicando destructuring. 

  
function fn1(
    { height=250, width=250, lineStroke='green'}) {
       //code
    }
