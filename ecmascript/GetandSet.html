Funciones: Getter and Setter


Getter

Nos sirve para obtener, recuperar o acceder al valor ya asignado de un atributo y utilizarlo para otra funcion. Su estructura es: 

            
   get name() {
      if(this._name) {
        return this._name;  
      } else {
        return undefined;
      }  
    }
Setter

Sirve para asignar un valor inicial a un atributo, pero de forma explícita, además el Setter nunca retorna un valor, y solo nos permite dar acceso público a ciertos atributos. Su estructura es: 

            
    set name(newName) {
      if (newName) {
        this._name = newName;
      } else {
        return false;
      }
    }

El siguiente ejemplo me permite obtener y asignar un nombre a traves del uso de los getter and setter. Ambas funciones estan declaradas dentro de la clase nombre. 

  
class nombres {

    constructor(name) {
        this.name = name;
    }

    get name() {
      if(this._name) {
        return this._name;  
      } else {
        return undefined;
      }  
    }

    set name(newName) {
      if (newName) {
        this._name = newName;
      } else {
        return false;
      }
    }
}

let nn = new nombres("damian");
