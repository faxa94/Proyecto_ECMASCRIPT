Funciones Vacias

Las funciones que aceptan parametros, producen errores si se inicializan sin ningun parametro. Para evitar este incoveniente se declara un arreglo vacio dentro como parametro de la funcion.

            
function fn(ee=[]){
  let leng= ee.length;
  console.log(leng);
}
fn('relampago'); //return 9
fn(); // return 0
Funciones con Objetos

Los paramtros de una funcion pueden ser variables tipo entero, decimal, caracter, funciones, arreglos y objetos. 

            
function fn(nombre, {cal1,cal2, calf3}={}){
  console.log("Nombre: ", nombre);
  console.log("Calificaciones: " ,cal1, cal2, calf3);
  
}
let ver={
"cal1":18,
"cal2":15,
"calf3":13
}
fn("pedro reyes", ver); //return pedro reyes, 18,15,13
fn("pedro reyes"); //return pedro reyes y undefined
Funciones Flecha

Utilizan los signos '=>' para denotar el valor que retorna la funcion. La declarar una funcion flecha se lo realiza mediante la siguiente expresion: 

            
var add = ( x, y ) => x + y;
console.log( add( 2, 4 ) ); // 6
console.log( add( 4, 4 ) ); // 8
console.log( add( 1, 2 ) ); // 3
