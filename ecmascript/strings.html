Template String

Las cadenas de caracteres se concatenaban en ES5 de la siguiente manera:

          
  function string(str1, str2, str3){
  let oracion= str1+" "+str2+" "+str3;
  return oracion;
}
console.log(string("rayos","y","lluvia"));

Se utilizaba el operador aritmentico (+) para unir variables, independientemente cual fuera su tipo

En ES6 se han agregado nuevo simbolos para la concatenacion de cadenas como son: 

  
    function string(str1, str2, str3){
  let oracion=`${str1} ${str2} ${str3}`;
  return oracion;
}
console.log(string("rayos","y","lluvia"));

Tambien se pueda dar uso con este formato a strings multilinea

  
    function string(str1, str2, str3){
  let oracion=`El ${str1} es una poderosa descarga natural de electricidad estática, 
producida durante una tormenta eléctrica, 
que genera un pulso electromagnético. 


La descarga eléctrica precipitada de la ${str3} 
es acompañada por la emisión de luz.`;
  return oracion;
}
console.log(string("rayo","y","lluvia"));
